# OnLineShop - второе задание по ASP
## Задание
### Подзадача 1. Создать модель для описания пользователя
1. Уникальный идентификатор
2. Фамилия Имя
3. Электронная почта
4. Фото/аватар (необязательное поле, но по умолчанию стоит картинка)
### Подзадача 2. Создать конроллер, где создается один пользователь и передается во View
### Подзадача 3. Создать View, отображающий информацию о пользователе с ипользованием bootstrap
## Порядок выполнения задания
1. создать форк этого репозитория
2. для выполнения задачи создать новую ветку
3. ветка должна называться фамилия_lesson_2_1
4. после выполнения подзадачи создать  Pull request с именем ветки
5. после получения approved сливаете ветку с заданием в главную ветку своего приложения
6. если вы получили комментарий об ошибке, нужно его исправить и сделать снова Push в свой репозиторий. 
Таким образом вы отправите свою работу на новое ревью



Это приложение мы будем доделывать до конца семестра, поэтому постарайтесь не набардачить)
